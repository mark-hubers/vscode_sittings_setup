üìö UI Parts Detailed Reference

1. Title Bar

Shows file name, folder, and window controls. On Mac, may include menu bar.

titleBar.border = your hot pink border

2. Activity Bar

Vertical icons on the left: Files, Search, Git, Debug, Extensions.

activityBar.border = hot pink

Toggle: Not recommended (you need those icons!)

3. Side Bar

File explorer, search results, git changes. Content changes based on Activity Bar selection.

sideBar.border = hot pink

Toggle: Cmd+B

4. Tab Bar

Shows all open files as tabs. Click to switch between files.

tab.activeBorder = hot pink

Close tab: Cmd+W

5. Breadcrumbs ‚≠ê

THIS IS WHERE YAML PATH SHOWS! Also shows file path and code structure.

For YAML: person ‚Ä∫ contact ‚Ä∫ email

For JSON: metadata ‚Ä∫ author ‚Ä∫ name

breadcrumbs.enabled: true (you have this on!)

6. Editor

Where you actually write code. Can split into multiple panes.

editor.background = pure black #000000

editorGroup.border = hot pink (for splits)

7. Panel

Terminal, Output, Problems, Debug Console. Usually at bottom.

panel.border = hot pink

Toggle: Cmd+J

8. Status Bar

Git branch, language, cursor position, errors/warnings. Clickable items!

statusBar.border = hot pink

Always visible (can't toggle off)

Editor Parts

Line Numbers: Left side, shows line numbers

editorLineNumber.foreground = bright cyan

Gutter: Breakpoints, git changes (left of line numbers)

Indent Guides: Vertical lines showing indentation
